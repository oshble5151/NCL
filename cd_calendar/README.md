# cd_calendar

Julian/Gregorian 날짜를 UT 참조 날짜로 변환해줌.

다양한 달력 시스템에 있을 수 있는 시간 값의 다차원 배열을 인수로 받는다.

## option

option을 통해 출력형식을 조절할수 있다.

__0 -> years__

__1 -> months__

__2 -> days__

__3 -> hours__

__4 -> minutes__

__5 -> seconds__
